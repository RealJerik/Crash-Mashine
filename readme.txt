Achtung!
Die Absturzmaschine ist ein Prototyp und kann extrem schweer aufgehalten werden!
Sie öffnet hunderte von Tabs in dem Standart-Web-Browser und könnte 
bei nicht sehr leistungsfähigen PC's Das System extrem verlangsamen!
Am besten einen der beigelegten Failsafe's bereithalten.

Failsafe-2 !!NUR WENN FAILSAFE-1 NICHT FUNKTIONIERT!! Schaltet den PC ohne Warnung sofort aus.

English:

Attention!
The crash machine is a prototype and can be stopped extremely sweltering!
It opens hundreds of tabs in the standard web browser and could
if the PC is not very powerful, slow down the system enormously!
It is best to have one of the enclosed failsafe available.

-------------------------------------------------- ------------------------------------------

Failsafe-1 Clears all tabs and closes the browser.

Failsafe-2 !! ONLY IF FAILSAFE-1 DOES NOT WORK !! Turn off the PC immediately without warning.
(Must be run as an administrator)
